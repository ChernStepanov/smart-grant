# Название проекта

Краткое описание проекта и его назначения.

## Требования

- Python 3.x
- Windows
- Зависимости из requirements.txt

## Установка

1. Клонируйте репозиторий или скачайте файлы проекта

2. Установите зависимости:

```
pip install -r requirements.txt
```

3. Создайте файл конфигурации `config.py` в корневой директории проекта со следующим содержимым:

```
NVIDIA_API_SC   = "ваш_nvidia_api_ключ_sc"
NVIDIA_API_BILL = "ваш_nvidia_api_ключ_bill"
```

**Важно:** Файл `config.py` содержит конфиденциальные API-ключи и не должен публиковаться в репозитории. Убедитесь, что он добавлен в `.gitignore`.

## Структура проекта

```
project/
│
├── config.py          # Файл конфигурации с API-ключами (создается вручную)
├── requirements.txt   # Зависимости проекта
├── main.py           # Главный файл (укажите ваш файл запуска)
└── README.md         # Этот файл
```

## Запуск

Запустите проект командой:

```
python main.py
```

## Конфигурация

В файле `config.py` необходимо указать ваши API-ключи NVIDIA:
- `NVIDIA_API_SC` - ключ для первого сервиса
- `NVIDIA_API_BILL` - ключ для второго сервиса
